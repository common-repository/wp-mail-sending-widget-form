== Description ==

> This Plugin will create a widget which will help you to create a submit form on the frontend of your website.
 	* The form will not only send an email with the infos of the subscribers to your email address but also save those infos on the plugin setting page on your dashboard.
	* Tested on Twenty Fourteen, Twenty Fifteen, Tweenty Sixteen (free wordpress theme) and the most popular AVADA theme.

== Installation ==

The Installation is very simple.

1. Upload the plugin files to the `/wp-content/plugins/wp-mail-plug` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the `Plugins` screen in WordPress.
3. Go to your widget Menu and Setup the "WP mail widget" named widget to any of your sidebar.
4. Finished!
